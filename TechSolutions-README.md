# TechSolutions Inc. — Microsoft 365 Enterprise Deployment

A self-directed Microsoft 365 deployment covering identity, security, collaboration, and monitoring — configured end to end in a live M365 tenant for a 300-employee Canadian organization.

**Live case study →** https://rahatislamanik-spec.github.io/TechSolutions-Microsoft365/

---

### What was built

A complete Microsoft 365 environment for TechSolutions Inc. — from bulk user onboarding to Insider Risk Management — deployed and verified across a live tenant.

### Four phases

| Phase | Focus | Key Configurations |
|---|---|---|
| 01 | Identity & Onboarding | Bulk CSV import, E5 licensing, M365 Groups, SharePoint permissions |
| 02 | Security & Protection | Defender, Safe Links, Safe Attachments, DLP, Insider Risk, Adaptive Protection |
| 03 | Collaboration & Governance | SharePoint sites, document libraries, OneDrive retention, Viva Engage |
| 04 | Monitoring & Automation | Audit logs, alert policies, Power Automate reports, Service Health |

### Technologies

Microsoft Entra ID · Exchange Online · SharePoint Online · OneDrive for Business · Microsoft Defender for Office 365 · Microsoft Purview · DLP · Insider Risk Management · Adaptive Protection · Viva Engage · Power Automate · Microsoft Graph API

### By the numbers

- **300 employees** onboarded across 3 departments
- **Microsoft 365 E5** licensing configured
- **42 screenshots** — every configuration verified in a live tenant
- **0** undetected data breaches — DLP live-tested and confirmed
